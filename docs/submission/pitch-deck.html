The Problem

Every SOC is an island

✕ CSV dumps

Threat intel shared as spreadsheets — no machine readability, no verification

✕ No provenance

"Who said this? When? From what data?" — currently answered by trust alone

✕ Vendor lock

Centralized platforms require trusting a vendor with cross-org data

✕ Sovereignty tradeoff

Collaboration vs. data sovereignty — treated as a binary choice

The Solution

Cortex Memory Fabric

📝 Publish

Agents produce signed memory articles with full provenance — source hash, DID, timestamp

🔍 Query

Semantic retrieval across org boundaries, ranked by hybrid cosine + trust score

🔗 Derive

New articles cite sources; trust propagates through the provenance graph

Ed25519 signatures on every article · Scope ACLs: private │ partner │ public · Trust formula: 0.6·base + 0.4·source − penalty
